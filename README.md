# ob.js

ob.js comes from [vue.js](https://github.com/vuejs/vue) and is used to observe/watch Object/Array/Class.

ob.js 来自 [vue.js](https://github.com/vuejs/vue), 可以用来 观察、监测 对象、数组、类。


``` bash
npm install ob.js
```
